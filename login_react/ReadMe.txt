This project is a simple login functionality.
1. we will be creating a homepage with signIn and SingUp button.
2. Click on signIn open a login screen where you can enter login credentials.
3. on submit if Authentication fails it will display a error message.
4. if Authentication successful it will navigate to homepage which will have a logout button.
5. We will also be saving the JWT token returned from server in the local storage so that 
    refreshing the screen does not require re authentication. 
6. on click of logout we will clear the local storage and navigate back to the initial home screen 
    with signin and signup buttons.
7. click on Signup will open a registration form where you can enter your details and click submit.
8. this shoudl display error message if signup failed , 
    else should display success message and redirect you to the login page.
